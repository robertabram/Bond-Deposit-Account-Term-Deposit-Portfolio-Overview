# Overview
The current dashboard provide the summary of Term Deposits, Current Deposits and Bonds Portfolio of Retail division.

# Business Objectives
Monitor portfolio movement
Compare actual balances to set targets
Analyze portfolio distribution across type of customers

# Type of Customers
Physical
Legal
Corporate

# Data model
Dim Tables : Deposits_Dim_By_Date, Funds_Involved_in_Bonds_Dim_By_Date
Fact Tables : Deposits_Fact, Funds_Involved_In_Bonds_Fact

# Tools Used
Microsoft SQL Server
Power BI
Excel
